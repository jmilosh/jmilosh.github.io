The first version of this simulation (shown in the [previous post](http://janmilosh.com/coating-line-simulation-using-d3-javascript-library/)) was created to help visualize the flow of items as they underwent a coating and curing process. This version is simply a revision with a more streamlined layout and different patterning for the items' movements. It is an animation, created with [D3.js](http://d3.js), a JavaScript library for data visualization.
